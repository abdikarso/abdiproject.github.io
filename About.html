Tentang Abdi Project

Abdi Project adalah sebuah usaha yang bergerak dibidang jasa konsultasi Automation.
Abdi Project dibentuk oleh Bapak La Ode Abdi Karso Panca Ramadhan. Pak Abdi mulai mempunyai ide untuk membuat Abdi Project bermula dari saat Pak Abdi belajar Programming dari Niomic.
Di Usia yang masih tergolong muda (21), Pak Abdi sudah memahami beberapa bahasa pemrograman Automation dan ditambah lagi dari kursus bersama Niomic.
Saat ini Abdi Project hanya melayani jasa konsultasi bidang Automation Machine, kedepannya Abdi Project bisa menjadi Software Developer Web maupun Mobile
